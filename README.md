# Web Slider – Bag Products Website

A responsive website for showcasing and selling bags (wholesale and retail): non-woven, PP, paper, and jute bags.

## Features

- **Products page** with Wholesale and Retail sections (PP Bags, Paper Bags, Non Woven Bags, Jute Bags)
- **Search** across all products
- **Owner-only product listing** – password-protected area to add new products (stored in browser)
- **Product detail modal** with images, specs, and features
- **Responsive layout** for mobile, tablet, and desktop
- **Header** with navigation and mobile drawer
- **Footer** with links and social icons

## Tech

- HTML, CSS, JavaScript
- No build step; open `index.html` or host the folder as a static site

## Usage

1. Open `index.html` in a browser or deploy the folder to any static host.
2. **Owner login:** Click **Owner** in the nav, enter the password (set in `app.js` as `OWNER_PASSWORD`), then use **+ Add** to list new products.
